# SkyXing 1.1.0 更新日志

> 发布性质：多平台客户端扩展（Multi-platform Clients）
> 覆盖平台：Web · Android · Windows

1.1.0 是平台的“多端扩展”版本，正式推出独立的 Android 与 Windows 客户端，并将全平台版本统一至 1.1.0。

## 一、新增功能

### Android 平台
- 新增独立 Android 客户端：React Native（Expo）+ React Navigation，采用底部 Tab 导航（主页 / 播客 / 我的）+ 栈导航。
- 客户端内置设置上下文（主题、调试模式、终端日志），与桌面端体验保持一致。
- Tauri 构建链路支持 Android 目标，可产出 APK 安装包。

### Windows 平台
- 新增独立 Windows 桌面客户端，基于桌面外壳 + React 前端，提供原生窗口体验。
- Tauri 构建链路支持 Windows 目标，可产出 NSIS 安装包。

### Web 平台
- 全站版本统一提升至 1.1.0（web / 根仓库 package.json 同步）。

## 二、改进项
- 将跨平台共享逻辑（API 客户端、HTML 净化器）抽离至 `clients/shared`，消除多端重复代码。
- 文章列表的标签筛选与搜索交互更加流畅，支持即时过滤。
- 统一各端的登录态与令牌管理，改善多端切换体验。
- 完善项目结构文档与本地开发 / 构建说明。

## 三、修复问题
- 修复多端登录态不同步问题（统一 API 客户端与令牌存储）。
- 修复部分机型上文章图片加载失败时的占位与异常提示。
- 修复管理后台在空数据状态下的崩溃与空白页问题。
