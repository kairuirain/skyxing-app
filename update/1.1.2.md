# SkyXing 1.1.2 更新日志

> 发布性质：稳定性与兼容维护（Stability & Compatibility）
> 覆盖平台：Web · Android · Windows

1.1.2 为 1.1.1 之后的稳定性与兼容性维护版本，重点解决多端在真实设备与系统版本下的运行问题，并完善发布构建流程。

## 一、新增功能
- Windows 客户端安装包增加版本自校验信息，便于安装后核对当前版本。
- Android 客户端补充安装来源与权限说明，提升首次启动引导的清晰度。

## 二、改进项
- Web 前端依赖与构建配置清理，降低体积并提升打包稳定性。
- Android 构建链路统一 SDK / NDK 路径探测逻辑，提升不同开发环境下的可构建性。
- Windows 安装流程文案与进度反馈优化，减少等待时的不确定性。
- 统一三端（Web / Android / Windows）的版本号定义，避免多渠道版本不一致。

## 三、修复问题
- 修复部分 Android 设备启动时的权限弹窗重复出现的问题。
- 修复 Windows 客户端在高 DPI 多显示器切换场景下的窗口位置错乱。
- 修复 Web 端偶发的静态资源 404（缓存哈希未刷新）问题。
- 修复后台统计接口在边界数据下的空值异常。
