# SkyXing 2.0.0 更新日志

> 发布性质：重大版本（Major Release）
> 覆盖平台：Web · Tauri APP (Windows/Android)

## 概要

2.0.0 是 Windows / Tauri 客户端的重要里程碑版本，对界面主题系统进行了彻底重构，并统一了更新发布流程，安装包纳入自研 OTA 体系，发布即生效。

---

## 新增功能

### 一、主题系统 2.0（深色模式重构）

- **跟随系统外观**：默认不再强制浅色，系统为深色时应用自动以深色呈现，无需手动切换。
- **修复浅色锁死**：旧版本会在首次渲染时把偏好写死为浅色，导致已设置深色的用户始终显示浅色；新版本仅在用户显式点击切换时才持久化，并保留历史深色意图。
- **显式设置标记**：引入 `theme_set` 标记，区分「用户主动设置」与「默认跟随系统」，避免旧 cookie 锁死。

### 二、通知系统

- 新增通知页面，集中展示系统通知与互动消息。
- 配套通知列表组件与后端通知路由。

### 三、统一更新发布流程

- Windows 客户端版本升级至 2.0.0，安装包纳入自研 OTA 体系。
- 发布说明以 `app/update/2.0.0.md` 维护，与 GitHub Release 自动同步。

---

## 问题修复

- 修复设置页未登录时错误显示退出按钮的问题。
- 修复多处深色模式覆盖不全的问题（导航栏、协议页等）。
- 修复 Worker SPA fallback 硬编码资源 hash 导致部署后白屏的问题。
- 修复主页因 `useSync` 配合每秒变化的时间戳版本号，导致无限重渲染、崩溃并持续刷新的问题（用 `useCallback` 稳定加载函数引用）。
