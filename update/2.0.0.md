# SkyXing 2.0.0 更新日志

> 发布性质：重大版本（Major Release）
> 覆盖平台：Web · Tauri APP (Windows/Android)

## 概要

2.0.0 是 SkyXing 客户端的重要里程碑版本。本版本对界面主题系统进行了彻底重构，统一了更新发布流程（安装包纳入自研 OTA 体系，发布即生效），并围绕「交互体验」「账号体系」「安全」三大方向带来大量新能力与修复。

---

## 新增功能

### 一、主题系统 2.0（深色模式重构）

- **跟随系统外观**：默认不再强制浅色，系统为深色时应用自动以深色呈现，无需手动切换。
- **修复浅色锁死**：旧版本会在首次渲染时把偏好写死为浅色，导致已设置深色的用户始终显示浅色；新版本仅在用户显式点击切换时才持久化，并保留历史深色意图。
- **显式设置标记**：引入 `theme_set` 标记，区分「用户主动设置」与「默认跟随系统」，避免旧 cookie 锁死。

### 二、通知系统

- 新增通知页面，集中展示系统通知与互动消息。
- 配套通知列表组件与后端通知路由。

### 三、统一更新发布流程（OTA 2.0）

- Windows / Android 客户端均升级至 2.0.0，安装包纳入自研 OTA 体系。
- 发布说明以 `app/update/2.0.0.md` 维护，与 GitHub Release 自动同步。
- **按操作系统区分下载**：OTA 根据客户端上报的系统参数返回对应平台的安装包；未识别/低版本客户端回退到 GitHub Release 最新页并提示手动下载。

### 四、入场动画与加载体验

- **按钮触发入场动画**：点击入口按钮时，从按钮图标位置精确放大铺满目标区域，返回时反向缩回按钮位置（取代旧的放大+模糊）。
- **二级菜单从右侧滑入**：`/account/*`、`/settings`、`/notifications`、`/privacy`、`/admin` 等子页面以全屏右侧滑入方式进入，覆盖整个应用（含导航栏）。
- **全局圆形加载动画**：将原加载动画统一替换为圆圈（Spinner）样式。

### 五、刷新交互

- Web / Windows：右下角圆形刷新按钮，随手可点。
- Android：下拉刷新（触摸手势 + Spinner）。
- 文章页新增「← 返回」按钮。

### 六、账号体系与权限重构

- **三级角色**：`user(1)` / `admin(2)` / `official(3)`，移除旧的 `author` 角色，统一显示为「用户」。
- **权限边界**：`admin` 仅可管理 `user`；`official` 可管理所有角色（含角色变更）。
- **SkyXing 官方账号**：首次请求时由后端幂等引导创建，用于发布官方内容。
- **条款与隐私**：
  - 「用户协议」重命名为「服务条款」（`/terms`），「隐私政策」（`/privacy`）。
  - 新增「账号所有权归属」与「非官方内容免责声明」条款。
  - App 端隐私/条款页直接读取线上 `skyxing.dpdns.org/privacy` 与 `/terms` 内容展示，保持与 Web 一致（Tauri 端通过 HTTP 插件绕过 WebView CORS）。

### 七、两步验证（2FA / TOTP）

- 基于 TOTP（HMAC-SHA1，Base32 密钥，otpauth URI）实现两步验证。
- 支持开启 / 验证设置 / 关闭，以及登录时的二次验证流程（返回 `requireTotp` + `tempToken`）。
- **二维码**：使用 `api.qrserver.com` 生成可扫描二维码。
- Web 端「设置」页同步接入 2FA 开关，状态由 `user.totpEnabled` 实时派生，开启后自动刷新用户信息。

### 八、全局头像显示与更新

- 新增通用 `Avatar` 组件，所有首字母占位统一替换为「头像图片 / 首字母渐变占位」：个人中心（MinePage / MyPage）、用户主页（UserPage）、文章作者卡片、评论列表（ArticlePage）均已接入。
- 账号信息（App 端 `/account/info`）与资料编辑（Web 端用户主页）支持**本地选择图片上传**：图片在前端压缩为 256px 的 JPEG data URL 后随资料保存，无需额外对象存储。
- 更新后通过 `updateProfile` / `refreshUser()` 全局同步，所有展示处即时刷新；Web / Windows / Android 三端通用。

## 问题修复

- 修复设置页未登录时错误显示退出按钮的问题。
- 修复多处深色模式覆盖不全的问题（导航栏、协议页等）。
- 修复 Worker SPA fallback 硬编码资源 hash 导致部署后白屏的问题。
- 修复主页因 `useSync` 配合每秒变化的时间戳版本号，导致无限重渲染、崩溃并持续刷新的问题（用 `useCallback` 稳定加载函数引用）。
- 修复非主页刷新显示 Not Found：改用 `not_found_handling = "none"` + Worker 回退到 `/index.html`（含 `no-store` / `CDN-Cache-Control: no-store` / `Clear-Site-Data` 等反缓存头）。
- 修复官方账号无法访问 `/admin` 的问题。
- 修复文章页 `ArrowLeft is not defined` 导致的崩溃（改用 `← 返回` 文本）。
- 修复 OTA 未区分操作系统、导致跨平台误装的问题。
- 修复 CDN 缓存持续返回旧版 UI 的问题（强反缓存头 + 主动清缓存）。
- 修复 App 端隐私/条款页因 WebView CORS 无法加载远程内容的问题（改用 Tauri HTTP 插件）。
- 修复 2FA 已开启但界面仍显示「未开启」的状态同步问题（状态从 `user` 派生并主动刷新）。

## 构建说明

```bash
# Windows 安装包（NSIS）
cd app
npm run tauri:build:windows

# Android APK / AAB
cd app
npm run tauri:build:android
```

> 构建产物：
> - Windows：`app/src-tauri/target/x86_64-pc-windows-msvc/release/bundle/nsis/SkyXing_2.0.0_x64-setup.exe`
> - Android：通过 `npx tauri android build` 生成 APK/AAB（需配置 Android SDK / NDK 与签名）。
