# SkyXing 1.2.5 更新日志

> 发布性质：正式版（Stable Release）
> 覆盖平台：Web · Tauri APP (Windows/Android)

## 概要

1.2.5 是一次重要的稳定版发布，包含了实时消息系统、安卓状态栏适配、管理员文章排序管理，以及多项 Bug 修复。

---

## 新增功能

### 一、实时消息系统

基于轮询 + 可见性检测的类 WebSocket 实时通信方案，无需 Durable Objects 即可实现近似实时的消息体验：

- **自动轮询**：会话页面 3 秒自动检查新消息，列表页面 5 秒
- **连接状态指示**：页面顶部显示 🟢 连接中 / 🟢 已连接 / 🔴 已断开
- **离线消息队列**：发送失败自动加入重试队列，5 秒后自动重发
- **已读回执**：已读消息显示 ✓✓，未读显示 ✓
- **可见性检测**：切回页面时立即刷新，离开时暂停轮询

### 二、管理员文章排序管理

- **多维度排序**：支持按发布时间、阅读量、权重三种条件排序
- **升降序切换**：点击排序标签切换升序/降序
- **自定义权重**：支持管理员直接编辑文章的权重值，实现灵活的自定义排序

### 三、安卓状态栏适配

修复安卓端应用启动后系统状态栏遮挡页面内容的问题。通过 CSS `env(safe-area-inset-top)` 和环境变量回退确保内容不被覆盖。

---

## 问题修复

### 1. Markdown 渲染修复

- 修复 `<pre><code>` 内 `&amp;lt;` 被二次转义显示为 `&amp;amp;lt;` 的问题
- 修复更新说明无法渲染 Markdown 的问题

### 2. OTA 版本比较

修复 `compareVersion` 无法识别 prerelease 后缀的问题，`1.2.5-beta.2 > 1.2.5-beta.1` 现在正确返回 `true`。

### 3. 跨渠道更新提示

修复测试版用户在稳定版渠道无更新提示的问题，跨渠道时强制 `hasUpdate: true`。

### 4. 链接跳转修复

- 文章内链接改用绝对 URL `https://skyxing.dpdns.org/link?url=...` 避免 Tauri 本地协议解析
- document 级事件委托 + Tauri `opener` 插件支持
