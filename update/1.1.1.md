# SkyXing 1.1.1 更新日志

> 发布性质：体验优化与稳定性维护（Maintenance & UX Polish）
> 覆盖平台：Web · Android · Windows

1.1.1 为 1.1.0 之后的体验优化与稳定性维护版本，聚焦于性能、交互细节与已知问题修复。

## 一、新增功能
- Windows 客户端引入设置面板雏形，集中管理终端日志、主题与调试开关入口。
- Android 客户端个人中心（我的）页交互与信息展示增强，整合资料卡、快捷操作与账户退出。

## 二、改进项
- Web 前端构建与首屏加载性能优化，减少不必要的重渲染。
- Android 客户端交互细节打磨：列表滚动顺滑度、按压反馈与图标可读性提升。
- Windows 客户端窗口初始化与缩放逻辑优化，提升高分屏（HiDPI）下的显示清晰度。
- 统一多端错误提示样式与空状态占位，降低用户困惑。
- 文章富文本渲染对代码块、引用、列表的样式进一步规范化。

## 三、修复问题
- 修复高分屏下 Windows 客户端窗口尺寸与 DPI 缩放不匹配导致的模糊问题。
- 修复文章分页在边界条件下（最后一页删除）的越界与空白问题。
- 修复 Android 端个别页面在刘海屏 / 手势导航区的安全区适配缺失。
- 修复评论提交后列表未即时刷新的状态同步问题。
- 修复 Web 端登录失效后重定向偶发的死循环。
