# SkyXing 1.1.5 更新日志

> 发布性质：实时通信优化、性能提升与 Bug 修复（Real-time Communication & Performance）
> 覆盖平台：Web · Android · Windows

1.1.5 聚焦通信体验优化：引入智能轮询实现近实时消息同步，优化网络请求策略，并修复若干影响日常使用的问题。

## 一、实时通信优化

- **智能轮询机制**：前端 3 秒轻量心跳轮询 `unread-count`，仅在 `latestUpdatedAt` 变化时自动加载全量数据，兼顾实时性与请求成本。
- **Visibility API 暂停**：标签页或窗口不可见时自动停止所有轮询，切回时立即检查——零无用后台请求。
- **手动刷新按钮**：私信列表与对话详情页增加刷新按钮，支持用户主动触发数据更新，刷新时带旋转动画反馈。

## 二、消息功能修复

- **修复名称显示错误**：会话详情页 header 使用对方用户名（`otherUser.displayName` / `username`），不再显示用户 ID。
- **修复消息不同步**：发送消息后乐观更新（立即追加到本地列表），配合 4 秒轮询确保双方实时可见。
- **修复对话详情获取优化**：后端 `GET /messages/conversations/:convId` 同时返回 `otherUser` 与会话信息，前端无需额外请求。

## 三、性能优化

- **列表并行批量获取**：文章列表/标签/评论 `kvGetMany` 并行，消除串行 N 次 KV 往返。
- **评论索引 O(1)**：`COMMENT_INDEX:<articleId>` 取代全量扫描，旧数据首访自动懒重建。
- **边缘缓存头**：文章列表（`max-age=60`）、标签（`max-age=300`）、评论（`max-age=30`），Cloudflare 边缘缓存降低源站压力。

## 四、安全加固

- **JWT_SECRET 外置**：从配置明文移除，改为 Cloudflare 加密 Secret 注入。
- **密码哈希 PBKDF2**：旧 SHA-256 升级为 PBKDF2-HMAC-SHA256（10 万次迭代 + 随机盐），登录自动迁移。
- **注册 O(1) 去重**：邮箱索引替代 `kvList` 全量扫描。

## 五、修复问题

- 修复 JDK 26 导致 Android 构建失败的问题。
- 修复 `@tauri-apps/plugin-shell` 缺失导致 Tauri 前端构建失败的问题。
- 修复评论索引 `parseJson` 错误导致 500 的问题。
