# SkyXing 1.0.0 更新日志

> 发布性质：初始版本（Initial Release）
> 覆盖平台：Web · Android · Windows

SkyXing 1.0.0 是平台的首次公开发布，包含完整的后端服务、Web 前端以及跨平台客户端雏形。

## 一、新增功能

### Web 平台
- 基于 React + Vite + TailwindCSS 构建的完整博客前端。
- 页面体系：首页文章流、文章详情、登录 / 注册、写文章 / 编辑文章、个人主页、管理后台、404 页。
- 文章中心：文章列表（分页）、标签筛选、关键词搜索、文章详情与富文本渲染。
- 用户系统：JWT 认证（jose），注册 / 登录 / 获取当前用户 / 个人资料更新。
- 互动能力：文章评论的创建、编辑、删除。
- 管理后台：概览统计、用户管理、文章管理三大模块。

### 后端服务（Cloudflare Worker）
- 基于 Hono.js + KV 存储的轻量后端，提供完整 REST API（认证、文章、评论、用户、管理统计）。
- 端点覆盖文章与评论的增删改查、标签聚合、用户与管理接口。

### Android 平台
- 跨平台客户端初始版本（Tauri + React），可浏览文章列表与登录。

### Windows 平台
- 跨平台桌面客户端初始版本（Tauri + React），提供桌面窗口承载应用。

## 二、改进项
- 引入统一的设计令牌（CSS 变量）体系，为后续主题与多端适配打下基础。
- Web 端采用响应式布局，兼顾桌面与移动浏览器访问。
- 抽象共享 API 客户端，减少前后端重复请求逻辑。

## 三、修复问题
- 实现 HTML 净化器（sanitizer），在渲染用户生成内容前过滤危险标签与脚本，修复潜在的 XSS 注入风险。
- 通过认证中间件统一校验 JWT，修复未授权访问受保护接口的越权问题。
- 修复文章 / 评论富文本中特殊字符导致的渲染异常。
